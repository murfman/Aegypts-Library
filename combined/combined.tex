\chapter{Introduction}
\lettrine[lines=4,findent=3pt]{T}{he} amount of references made within the entirety of John Crowley's Ægypt Tetralogy is rather infamous among a certain subset of readers. This is an on-going project to list the real and fictional books that are mentioned therein. So far this is a personal document that combines the bibliographies from other projects made for each book. I did this mostly so I didn't have to search separate files to see if a reference is made. 

For that slim minority that cares, I'm doing this in \LaTeX{}, well really it is \hologo{LuaLaTeX}, with \hologo{BibTeX} and multiple keywords. Essentially, I'm setting up filters using \textsc{defbibfilter} of multiple keywords and then displaying them each as a separate \textsc{printbibliography} command.
