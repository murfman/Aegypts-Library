\chapter{Introduction}
\lettrine[lines=4,findent=3pt]{T}{his} is a continuation of my attempt at listing out the various authors and books mentioned within the second volume of the Ægypt Tetralogy, Love \& Sleep, by John Crowley. Fictitious authors and books have been placed in bold typeface for easy identification. 

For those that care, this project was originally done in RStudio to make use of their excellent Rmarkdown code and integration abilities, especially those concerned with BibTeX files. Once I finished it there a friend mentioned it would be cool to have as a zine instead of a flat PDF, so I went way down that rabbit-hole before coming back to sanity. Well sort of\dots I'm using \LaTeX{} because I'm familiar enough with it and it is free and ridiculously powerful. So, after much trial and tribulation what started out as a text file then migrated into Markdown then Rmarkdown is now in \LaTeX{}, and from letter size to half-letter zine, and now back to letter, which I'm sure highlights my own inability to commit more than anything else.
