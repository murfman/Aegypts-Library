\chapter{Introduction}
\dropcap{T}{he} goal I set for this project is to list all of the books and authors mentioned throughout \textit{Ægypt}, or \textit{The Solitudes}, by chapter. A full bibliography for all items found is at the end of the document. If a chapter is missing below, then it did not have any books or authors mentioned within it. Books listed in bold are fictional works created by Crowley. The Authors Mentioned subsections list authors that were mentioned within a chapter but did not mention a particular book.

In most cases I tried to cite a work that would have been appropriate for the time frame of the story in which it is mentioned. In some cases, I have not been able to find a suitable edition and so cite a modern edition instead. Those will eventually be cleaned up to reflect either a copy of the work from the 16th century, one up to 1978, or one available to Crowley while he was writing.

This is probably the fourth or fifth version of this that I've done. The previous versions were for my the enjoyment of myself and a few others. This version I've tried to get as good as possible. I also made use of a {BiBTeX} file using the excellent \href{https://www.jabref.org/}{Jabref} application which I use in conjunction with \href{https://www.xm1math.net/texmaker/}{TexMaker}. \textit{(Earlier versions of this were entirely text or Markdown.)} 

I'm open to any ideas as to how this can be updated and expanded. For any corrections or comments the reader can email me directly at murfman@pm.me.

Special thanks goes to the guys within the \#Books channel on \href{https://www.mu-podcast.com}{The Miskatonic University Podcast} \href{https://discord.gg/5nH7UqtpdM}{Discord Server}. You guys know who you are.

\href{https://aegyptbibliography.substack.com}{Ægypt Bibliography} is an occasional blog regarding this project is available on \href{www.substack.com}{Substack}.
